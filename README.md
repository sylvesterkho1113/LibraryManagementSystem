# LibraryManagementSystem
Java-based graphical user interface (GUI) application designed as a  library management system


## System Features
- Login
- Add Book
- View Book
- Delete Book
- Update book's information
- Add Users
- View Users
- Borrow Books
- Return Books
- View Borrowing History
