# 📚 Library Management System

A robust desktop application built with **Java Swing** designed to streamline library operations. This system allows administrators to manage books, users, and borrowing records through an intuitive graphical user interface.

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Swing](https://img.shields.io/badge/Swing-GUI-blue?style=for-the-badge)

---

## ✨ Features

- **🔐 Secure Authentication:** Admin login system to protect library data.
- **📖 Book Management:** Add, update, delete, and browse the library's book collection.
- **👥 User Management:** Register and manage library members with ease.
- **🔄 Transaction Tracking:** Efficiently handle book issuance and returns with real-time status updates.
- **💾 Data Persistence:** Uses localized text files (`.txt`) for reliable data storage without the need for a complex database setup.
- **🖥️ Responsive UI:** Built with Java Swing for a consistent desktop experience.

- **Language:** Java
- **UI Framework:** Java Swing / AWT
- **Storage:** File-based persistence (`books.txt`, `users.txt`, `records.txt`)

## 🚀 Getting Started

### Prerequisites
- **JDK (Java Development Kit)** 8 or higher.
- A terminal or an IDE (IntelliJ IDEA, Eclipse, VS Code).

### Installation & Execution

1. **Clone the repository:**
   ```bash
   git clone https://github.com/chwankai/Library-Management-System.git
   cd Library-Management-System
   ```

2. **Compile the project:**
   ```bash
   javac *.java
   ```

3. **Run the application:**
   ```bash
   java LibraryManagementSystem
   ```
