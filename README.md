# 📚 Library Management System

## 🚀 Features

- **Book Management:** Add, update, remove, and search books
- **User Management:** Register, update, remove, and search library users
- **Borrow Records:** Track which users borrowed which books, including due dates and returns
- **User-friendly GUI:** Interact with the system easily via GUI

## 💻 Getting Started

### Prerequisites
* A Java compiler

### Installation & Execution
1. Clone this repository to your local machine.
   ```sh
   git clone https://github.com/chwankai/Library-Management-System.git
   cd Library-Management-System
   ```
2. Compile the program using your IDE.
3. Run the LibraryManagementSystem.java
